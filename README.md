# 소비 캐릭터 진단 MVP

3캐릭터 버전입니다.

## 포함 캐릭터

- 지름신 여우
- 소확행 햄스터
- 감정소비 고양이

## 적용 방법

GitHub 저장소 루트에 압축을 푼 뒤 전체 파일을 업로드하세요.

교체 대상:
- `index.html`
- `package.json`
- `src/App.jsx`
- `src/main.jsx`
- `src/styles.css`
- `src/assets/fox.png`
- `src/assets/hamster.png`
- `src/assets/cat.png`

## 실행

```bash
npm install
npm run dev
```
