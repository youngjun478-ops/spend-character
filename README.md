# 소비 캐릭터 진단 MVP

## 적용 방법

GitHub 저장소 루트에 이 압축 파일 안의 내용을 드래그앤드롭하세요.
기존 `src/main.jsx`, `src/styles.css`, `package.json`, `index.html`은 새 파일로 교체됩니다.

## 포함된 캐릭터

- 지름신 여우: 쇼핑/충동 중심
- 소확행 햄스터: 카페/간식/식비 중심

현재는 캐릭터당 표정 1개 전략입니다. 상태 차이는 이미지가 아니라 결과 문장과 카드 UI로 표현합니다.

## 실행

```bash
npm install
npm run dev
```
